<b>Tripie(트리피)는 다음과 같은 기능을 제공합니다.</b>

- `비로그인` : 트리플에서 웹스크래핑한 여행 지역에 대한 팁 (여행 지역, 일정, 장소 및 음식점 정보)
- `로그인` : github 또는 카카오로 로그인 시 챗지피티 기반의 여행 일정을 추천받을 수 있습니다.
